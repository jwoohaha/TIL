## MARKDOWN

- 제목
  
  - 글자 크기를 위해 제목 태그(#) 를 사용하면 안됨
  
  - 문서 구조를 잡아주는 역할
    
    

- 리스트 (* or -)
  
  

- 코드블럭 (백틱)
  
  문장을 작성하다가 ` 사용

```
print(test)
```

 3개 사용 시 multiline

```
print(test)
print(test)
print(test)
```



- 이미지
  
  - ![string](img_url

- 글자효과

## Linux

- tab : 자동완성

- mkdir : 디렉토리 생성

- touch : 파일생성

- ls : 내부 보여줌

- rm : 삭제

- cd ~ : root directory (절대경로)

- ./ : 현재 위치 (상대경로)

- ../ : 부모 디렉토리

- find -name 777 : 777이라는이름을 가진 폴더 혹은 파일 찾기
